# 📄 Propostas de Mudança (RFC do Dominó)

Aqui é onde a história é escrita.

Se você acredita que descobriu uma falha estrutural no dominó,
é aqui que você prova.

---

# Modelo Oficial

Crie um arquivo:

RFC-XXXX-nome-curto.md

Use este formato:

---

# RFC-XXXX - Nome da Regra

## Status
Proposed | Approved | Rejected

## Problema

Descreva a falha sistêmica detectada.

Evite:
- “Eu perdi.”
- “Não gostei.”

---

## Proposta

Descreva objetivamente a mudança.

Se precisar de 3 parágrafos,
talvez seja complexo demais para dominó.
