# 🧮 Sistema de Pontuação

Porque ganhar não é suficiente.
É preciso quantificar.

---

# 1. Quando Alguém Bate

- Soma-se o valor das peças restantes dos adversários.
- O vencedor recebe essa soma.

Exemplo:
Se restaram 14 pontos na mão dos adversários,
o vencedor ganha 14 pontos.

Simples.
Não complique.

---

# 2. Quando o Jogo Trava

- Soma-se o valor das peças de cada jogador.
- Vence quem tiver menor soma.
- Recebe a diferença entre sua soma e a dos adversários.

---

# 3. Erros de Cálculo

Se alguém errar a soma:

- Revisar imediatamente.
- Usar calculadora se necessário.
- Não discutir aritmética básica por 20 minutos.

Dominó é estratégia.
Não é teste de matemática do ensino fundamental.
